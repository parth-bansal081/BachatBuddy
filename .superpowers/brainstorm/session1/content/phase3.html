Phase 3 — AI & Atmosphere

3D AI Assistant + Immersive Environment Effects

New Components

AIAvatar
Friendly 3D avatar for financial assistant
• Simple geometric character (sphere head + body)
• Floating idle animation (gentle bob + rotation)
• Mouth/speech indicator when AI is "thinking"
• Points/looks toward relevant UI elements
• Appear/disappear with scale + fade transition
• Accessible: all tips also rendered as text
<AIAvatar
  state="speaking"
  tip={aiTip}
  position={[2, 0, 0]}
/>
DynamicLighting
Ambient lighting that responds to context
• Warm morning tones, cool evening tones
• Intensity pulses on data updates
• Spotlight follows cursor position
• Red-shift when over budget, green when saving
• Shared via useThreeTheme hook
ParallaxBackground
Depth-aware background planes
• Multiple layers at different Z-depths
• Each layer shifts at different rate on scroll
• Subtle gradient meshes with Bachat green
• Optional: low-poly city silhouette shapes
• Performance: static planes, no physics
CursorRipple
Interactive ground plane with ripple effect
• Flat plane below all 3D content
• Vertex displacement shader on cursor hover
• Ripples fade over time (decay)
• Color: subtle Bachat-green tint
• Opt-out via prefers-reduced-motion
