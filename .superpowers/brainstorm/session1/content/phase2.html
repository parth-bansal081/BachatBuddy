Phase 2 — Data Storytelling

Transaction Flow + Budget Orbs + Goal Towers

New Components

TransactionFlow
3D ribbon through space representing money flow
• TubeGeometry along a spline curve
• Tube thickness = transaction amount
• Each segment colored by category
• Hover: show tooltip with details
• Scrub-through time via scroll/drag
• Floating transaction "nodes" on the ribbon
<TransactionFlow
  transactions={txns}
  timeRange="month"
/>
BudgetOrb
Floating 3D spheres for budget categories
• Orbit around a central point
• Size = budget allocation %
• Color shifts from green → amber → red
• Pulse animation when near limit
• Click: expand to show details
• Drag: adjust budget allocation
<BudgetOrbSystem
  budgets={categories}
  radius={2}
/>
GoalTower
3D towers representing savings goals
• Stacked box geometry, each segment = milestone
• Height = % progress toward goal
• Unbuilt segments shown as wireframe
• Particles orbit completed milestones
• Camera zooms in on click for details
• Animated construction on progress update
<GoalTower
  goal={savingsGoal}
  showMilestones
/>
ThreeLineChart
3D time-series line chart
• Extruded line along time axis (X)
• Y = value, Z = category (depth layers)
• Gradient fill below the line
• Animated draw-in on mount
• Data point spheres at key events
• Replaces ProjectedSpendingChart
<ThreeLineChart
  series={projections}
  timeRange="6m"
/>
