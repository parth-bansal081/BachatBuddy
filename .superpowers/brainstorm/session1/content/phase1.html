Phase 1 — 3D Foundation Components

Shared library + Dashboard enhancement

Component Library (src/components/three/)

ThreeScene
Wrapper for every R3F Canvas
• Responsive sizing (fills parent)
• Camera defaults + controls (OrbitControls)
• Bachat-green ambient lighting
• Error boundary + WebGL fallback
• Performance monitor (FPS, auto-downgrade)
<ThreeScene>
  <BarChart data={...} />
  <ParticleField />
</ThreeScene>
ThreeBarChart
Animated 3D bar chart replacing Recharts
• Extruded bars with rounded caps
• Color = category, height = amount
• Hover: bar lifts + glows
• Label planes floating above each bar
• Props: data[], width, height, theme
<ThreeBarChart
  data={spendingData}
  width={600} height={300}
/>
ParticleField
Ambient floating particle system
• Shader-based GPU particles (thousands)
• Gentle drift + cursor attraction
• Color palette: bachat-green gradients
• Optional data-driven: particles = transactions
• Props: count, spread, speed, cursorReactive
ThreeRingChart
3D donut/ring for category breakdown
• Extruded torus segments (TorusGeometry)
• Each segment = category, height = %
• Hover: segment extends outward
• Center: total amount text (Html overlay)
• Animated entrance (segments rise in sequence)
