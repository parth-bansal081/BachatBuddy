BachatBuddy
Your Automated Financial Companion for the UPI Era.

BachatBuddy is a smart personal finance manager designed for the Indian market to eliminate "manual entry fatigue." By leveraging the Account Aggregator (AA) framework, it provides secure, zero-friction tracking of your financial life.

🚀 Live Demo
Access the application here: https://bachatbuddy-hq.vercel.app

✨ Key Features
Zero-Friction Tracking: Automated, consent-based data fetching via the AA ecosystem—no manual typing required.

Smart AI Categorization: Automatically sorts messy UPI IDs into clear categories like Food, Travel, and Rent.

Daily Safe-to-Spend: Calculates real-time spending limits based on your monthly income and savings goals.

Privacy First: Built with Row Level Security (RLS) to ensure your financial data is isolated and encrypted.

🛠️ Tech Stack
Frontend: Next.js, TypeScript, Tailwind CSS, Shadcn/UI

Backend: Supabase (PostgreSQL) with Row Level Security

State Management: Zustand / Redux Toolkit

Data Visualization: Recharts

🌍 Social Impact
BachatBuddy is committed to SDG 8 (Decent Work and Economic Growth) by promoting financial literacy and inclusion through modern digital infrastructure.
